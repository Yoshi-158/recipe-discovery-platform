# Recipe Discovery Platform

## Overview

Recipe Discovery Platform is a responsive web application that allows users to search and explore meal recipes using the MealDB API. Users can view recipe details including ingredients, cooking instructions, and meal images through a clean and interactive interface.

## Features

- Search recipes by meal name
- View recipe ingredients and instructions
- Dynamic content rendering using JavaScript
- Real-time API integration
- Responsive user interface
- User-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- MealDB API
- Fetch API

## Key Concepts Implemented

### Fetch API
Used to retrieve recipe data from the MealDB API.

### Asynchronous JavaScript
Implemented using async and await for efficient data fetching.

### DOM Manipulation
Dynamically updates the webpage with recipe information.

### Event Handling
Captures user search queries and displays matching recipes.

## How It Works

1. User enters a meal name.
2. Application sends a request to the MealDB API.
3. Matching recipes are retrieved.
4. Recipe details are displayed dynamically.
